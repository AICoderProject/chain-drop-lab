# TypeScript + Vite + Canvasで完全オリジナル連鎖パズルゲームを作ってみた

こんにちは！今回は、ブラウザでサクサク動くオリジナルの落ち物連鎖パズルゲーム **『chain-drop-lab (チェイン・ドロップ・ラボ)』** を開発し、MITライセンスのオープンソースとしてGitHubに公開した話を共有します。

* **GitHub リポジトリ**: [https://github.com/AICoderProject/chain-drop-lab](https://github.com/AICoderProject/chain-drop-lab)
* **Live Demo (今すぐ遊べます)**: [https://aicoderproject.github.io/chain-drop-lab/](https://aicoderproject.github.io/chain-drop-lab/)

---

## 🔬 プロジェクトの特徴

このゲームの最大の特徴は、**「商標・アセットのトラブルが一切ない、クリーンなOSSパズル」**である点です。
既存パズルゲームの商標（例：「ぷよぷよ」など）に依存せず、画像や音声ファイルを一切読み込まない仕様になっています。

### 主要スペック

| 技術領域 | 採用スタック | 特長 |
| --- | --- | --- |
| 言語 | TypeScript | 強力な静的型定義で堅牢なゲームループを実現 |
| ビルド | Vite + tsc | 高速なHMR（ホットリロード）とビルド |
| グラフィックス | HTML5 Canvas (2D) | パーティクルや画面揺れなどを動的数式描画 |
| サウンド | Web Audio API | リアルタイム波形合成による procedural 効果音 |
| 品質保証 | ESLint + Vitest | 盤面連結消去判定を100%テスト |

---

## 🛠️ 技術的アプローチの解説

### 1. アセットロード時間 0秒のベクターアニメーション
ゲーム中のジェム、連鎖発生時の極彩色パーティクル、カメラ揺れ、3D風浮遊テキストなどの演出はすべて、CanvasのJavaScriptコードで毎フレーム動的に数式計算してレンダリングしています。アセットのロード待ちは存在しません。

### 2. Web Audio APIによる連鎖ピッチシフトシンセ
効果音ファイル (mp3/wav) を使用せず、Web Audio APIでオシレータとゲインノードを作成し、リアルタイムにシンセ音声を合成しています。連鎖が進むごとに「ド、レ、ミ、ファ...」と音程が上がっていく仕様もプログラムコードだけで完結しています。

### 3. Vitest によるパズルロジックの完全テスト
連鎖パズルでありがちな「浮いたジェムが下に落ちない」「正しく連結が消去されない」などのバグを確実に防ぐため、盤面操作や DFS（深さ優先探索）による接続チェックといった純粋なロジック層を完全に切り離し、Vitestで網羅的なユニットテストを実施しています。

## 🚀 最新アップデート v1.0.2

最新のリリース **v1.0.2** では、GitHub Pagesのデモで初めて遊ぶユーザーが迷わないよう、操作方法を表示する **フローティング操作ガイドパネル** を追加しました。

- **PCビュー**: 右下に半透明（Glassmorphism）デザインでスッキリ配置
- **モバイルビュー (最大幅900px)**: 画面サイズ390pxでも横スクロールが発生しないよう、自動的に画面下部へインラインスタックされるレスポンシブCSSを適用
- **フォーカスセーフ**: ゲームCanvasのキーボードフォーカス入力を奪わない安全な設計
- **既存仕様の維持**: ESLintチェック、Vitest（17/17件）のテスト、および元のキーバインド仕様（Z/Xで回転、Space/Enter/Pで開始・ポーズ）を壊さずにインテグレーション完了
