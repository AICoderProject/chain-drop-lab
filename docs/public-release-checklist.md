# GitHub 公開前チェックリスト

本プロジェクトをGitHubに安全かつクリーンに公開するためのチェックリストです。

## 1. 知的財産・商標リスクの確認
- [ ] ゲーム内に「ぷよぷよ」「Puyo Puyo」「SEGA」などの他社商標やロゴが一切含まれていないこと。
- [ ] 他社の公式画像や効果音、音楽ファイルなどの無断使用が一切ないこと。（本プロジェクトは Canvas による自作描画、Web Audio API による自作SEに統一されています）
- [ ] LICENSE_PENDING.md および LICENSE ファイルが整備されていること。

## 2. セキュリティおよび機密情報の漏洩防止
- [ ] APIキー、プライベートトークン、暗号鍵などの機密情報（Secrets）がコード中にハードコードされていないこと。
- [ ] `.env` ファイルなどの環境変数ファイルや、開発用のテンポラリファイルが `.gitignore` によって適切に除外されていること。
- [ ] `git remote add` や `git push` をローカルの開発環境で手動で誤って実行していないこと。

## 3. テストと品質保証
- [ ] `npm run test` (Vitest) を実行し、すべてのロジックテスト（消去、連結、落下、スコア）がパスすること。
- [ ] `npm run lint` が警告なしでパスすること。
- [ ] `npm run build` がエラーなく完了すること。
- [ ] さまざまなブラウザ（Chrome, Firefox, Safari, Edge）でゲームが期待通り動作し、キーボード・マウス・タッチで問題なく操作できること。

## 4. GitHub Pages への対応
- [ ] `vite.config.ts` で `base` パスが適切に設定されていること。（リポジトリ名に合わせた `./` もしくは `/chain-drop-lab/` の設定）
- [ ] 静的ビルド成果物が `dist` に正しく出力され、余計なエラーを出さずにインデックスページが開くこと。
