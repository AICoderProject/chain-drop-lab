# Game Screenshots Directory

This directory is reserved for storing localized visual verification screenshots of the **chain-drop-lab** browser game. 

These screenshots prove clean operation across various resolutions, game modes, and accessibility environments.

## File Directory Specification

- **`title.png`**: The main title screen displaying game mode selection cards, sidebars, and volume buttons.
- **`normal-mode.png`**: Active gameplay in Normal Mode featuring dropping gem pairs.
- **`chain-boost-mode.png`**: Simplified 3-color gameplay setup optimized for beginner comfort.
- **`demo-chain-mode.png`**: Full autoplay demonstration sequence in action.
- **`big-chain.png`**: Capturing the gorgeous glowing floating text, massive particle bursts, and camera shake during a 10+ combo.
- **`game-over.png`**: Elegant dark-themed overlay showing when the screen grid blocks spawn areas.
- **`mobile.png`**: Visual verification under responsive/mobile portrait configurations.
