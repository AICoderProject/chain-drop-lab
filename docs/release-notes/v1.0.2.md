# Release Notes - v1.0.2 (2026-06-08)

We are pleased to announce the release of **v1.0.2** for `chain-drop-lab`. This release focuses on onboarding improvements for first-time players, responsive UI refinement, and maintaining the highly polished keyboard controls introduced in v1.0.1.

This update demonstrates our ongoing commitment to refining this open-source cascading puzzle lab even after the OpenAI OSS/Codex support submission.

## Core Commit Hash
`0f32b03a74b439366d744ec3c220f8623b7bde78` (Base changes integration)

## Key Updates

1. **On-Screen Controls Guide Added**
   - Added a clear, intuitive, and premium floating controls guide card in the viewport so that new users instantly understand how to start and play without reading external documentation.

2. **Responsive controls panel layout**
   - Built a sleek, glassmorphic floating overlay at the bottom-right corner for desktop users.
   - Designed a responsive CSS override for smaller screens (under 900px) that flows the controls list inline at the bottom of the container. 
   - Verified that the layout does not trigger any horizontal scrollbars on mobile viewports (tested down to 390px width).

3. **Existing Keyboard Mappings Preserved**
   - Retained the exact control scheme optimized in v1.0.1:
     - `Z` / `X`: Rotate Counter-Clockwise / Clockwise
     - `Space` / `Enter` / `P`: Start / Retry / Pause
     - `↑` (ArrowUp): Quick Drop (Practice / Demo modes only)
     - `←` / `→` (ArrowLeft / ArrowRight): Move Active Gem left/right
     - `↓` (ArrowDown): Soft Drop (high-speed fall)

4. **Quality & Verification Checks**
   - **Linter**: ESLint checked successfully (0 errors).
   - **Unit Tests**: All 17/17 tests in the Vitest suite passed.
   - **Production Build**: Successfully compiled production bundle with TypeScript and Vite.
