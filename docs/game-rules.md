# chain-drop-lab ゲームルール解説

## 概要
`chain-drop-lab` は、上部から2個1組で落下してくる色付きの「ジェム」を操作し、盤面に配置していく落ち物連鎖パズルゲームです。

## 基本ルール
1. **ジェムの配置**:
   - ジェムは赤、青、緑、黄、紫の最大5種類の色があります。
   - 2個セット（軸となるジェムと、その周囲を回るジェム）で落下してきます。
2. **消去条件**:
   - 同じ色のジェムが縦・横に **4個以上** 連結すると消滅します（斜めは連結しません）。
3. **重力落下と連鎖**:
   - ジェムが消滅すると、浮いたジェムが重力によって下まで落下します。
   - 落下した結果、再び同じ色のジェムが4個以上連結すると、さらなる消滅が発生します。これが **「連鎖（Chain）」** です。
   - 連鎖が起きるたびに、獲得スコアが倍増し、連鎖数に応じた派手なエフェクトが発生します。

## 操作方法
- **キーボード**:
  - `ArrowLeft` / `ArrowRight` : 左右移動
  - `ArrowDown` : 高速落下 (ソフトドロップ)
  - `ArrowUp` : クイックドロップ (一瞬で落とす、Practice / Demo Mode のみ)
  - `Z` : 反時計回り回転
  - `X` : 時計回り回転
  - `Space` : ゲーム開始 / リトライ / 一時停止解除
- **マウス / タッチ操作**:
  - 画面左右のタップ・クリック : 左右移動
  - 画面下部のタップ・クリック : 高速落下
  - 画面中央のタップ・クリック : 回転

## 得点（スコア）計算
消去したジェムの数と連鎖数に応じてスコアが計算されます。
$$Score = (消去ジェム数 \times 10) \times (連鎖ボーナス \times 色数ボーナス)$$
- 連鎖数が大きくなるほど、スコアの倍率が指数関数的に増加します。

## ゲームモード
- **Normal Mode**: 通常の難易度でハイスコアを目指すモード。
- **Chain Boost Mode**: 降ってくるジェムの色数が制限されるか、特殊な配色で連鎖が極めて発生しやすくなる初心者向けの「超連鎖体験モード」。
- **Practice Mode**: ゲームオーバーがなく、いつでも巻き戻しや自由な配置のテストができる練習用モード。
- **Demo Chain Mode**: コンピュータが自動で美しい超大連鎖を実演する鑑賞・学習用モード。
