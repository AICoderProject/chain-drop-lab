🔬 モード説明

【Normal Mode】
5つの色付きジェムを落とし、同じ色を4つ以上つなげて消す標準モード。ハイスコアを目指しましょう！

🎮 操作方法

  • ← / →: ジェムの左右移動
  • ↓ (ArrowDown): 高速落下 (Soft Drop)
  • Z / X: 左回転 / 右回転 (Rotate)
  • Space / Enter / P: 開始 / リトライ / ポーズ (Start / Retry / Pause)
  • ↑ (ArrowUp): 即時落下 (Quick Drop / Practice / Demoのみ)

※スマホ・タブレットは画面の左右タップで移動、中央タップで回転、下部タップで高速落下します。
